As API usage

APIs

GET /list-todos.cgi
List register TODOs.
GET /get-todo-by-id.cgi?ID
Get a to-do with specified ID. The return can be 200 or 404.
POST /add-todo.cgi
Register new to-do. When 200, the one column with such to-do is return.

Data format

Those APIs shall respond each to-do's as a record-oriented (separated by LFs) text, whose each row represents each to-do. 1st and 2nd items are delimined by one or more spaces and tabs. 2nd and 3rd items are delimined by exactly one space.

1st item: ID

2nd item: timestamp (YYYY-mm-dd_HH:MM:SS) when registered

3rd item: content of to-do (LF and backslash are represented as "\n" and "\\". Can include spaces and tabs.
